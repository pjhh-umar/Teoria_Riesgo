Ejercicio

En los siguientes ejercicios se dan funciones de costo total, donde $c$ es el costo de producir $q$ unidades de un producto. Para cada caso encuentre la función de costo marginal. ¿Cuál es el costo marginal para el valor o valores dados de $q$?

  1. \(c = 500 + 10q;\ q = 100\)

  2. \(c = 5000 + 6q;\ q = 36\)

  3. \(c = 0.1q^2 + 3q + 450;\ q = 5\)

  4. \(c = 0.1q^2 + 3q + 2;\ q = 3\)

  5. \(c = q^2 + 50q + 1000;\ q = 15, \ q = 16, \ q = 17\)

  6. \(c = 0.04q^3 - 0.5q^2 + 4.4q + 7500;\ q = 5, \ q = 25, \ q = 1000\)

Ejercicio

En los siguientes problemas $\overline{c}$ representa el costo promedio por unidad, que es una función del número $q$ de unidades producidas. Encuentre la función de costo marginal y el costo marginal para los valores indicados de $q$.

  1. \(\bar{c} = 0.01q + 5 + \displaystyle{\frac{500}{q}}; \ q = 50, \ q = 100\)

  2. \(\bar{c} = 2 + \displaystyle{\frac{1000}{q}}; \ q = 25, \ q = 235\)

  3. \(\bar{c} = 0.00002q^2 - 0.01q + 6 + \displaystyle{\frac{20000}{q}}; \ q = 100, \ q = 500\)

  4. \(\bar{c} = 0.002q^2 - 0.5q + 60 + \displaystyle{\frac{7000}{q}}; \ q = 15, \ q = 25\)

Ejercicio

  1. \(r = 0.8q;\ q = 9, \ q = 300, \ q = 500\)

  2. \(r = q\left(15 - \displaystyle{\frac{1}{30}}q\right);\ q = 5, \ q = 15, \ q = 150\)

  3. \(r = 250q + 45q^2 - q^3;\ q = 5, \ q = 10, \ q = 25\)

  4. \(r = 2q(30 - 0.1q);\ q = 10, \ q = 20\)
