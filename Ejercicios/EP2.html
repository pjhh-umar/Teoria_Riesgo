Ejercicio

Si el riesgo $S$ en el modelo colectivo tiene distribución $\mathrm{Bin\, Comp}(n,p,G)$. Prueba que

  1. $\mathbb{E}[S] = np\mathbb{E}[Y]$

  2. $\mathbb{E}\left[S^{2}\right] = np \left( \mathbb{E}\left[Y^{2}\right] + (n-1) p \left(\mathbb{E}\left[Y\right]\right)^{2} \right)$

  3. $\mathrm{Var}[S] = np \left( \mathbb{E}\left[Y^{2}\right] - p \left(\mathbb{E}\left[Y\right]\right)^{2} \right)$

  4. $\mathrm{M}_{S}(t) =\displaystyle{ \left(\displaystyle{ 1 -p + p\mathrm{M}_{Y}(t)} \right)^{n}}$

Ejercicio

Si el riesgo $S$ en el modelo colectivo tiene distribución $\mathrm{Bin\, Neg\, Comp}(r,p)$. Prueba que

  1. $\mathbb{E}[S] = r \left( \displaystyle{\frac{1}{p} -1} \right) \mathbb{E}[Y]$

  2. $\mathbb{E}\left[S^{2}\right] = r \left( \displaystyle{\frac{1}{p} -1} \right) \left[ \mathbb{E}\left[Y^{2}\right] + (r+1)\displaystyle{\left(\frac{1}{p} -1\right)} \left(\mathbb{E}\left[Y\right]\right)^{2} \right]$

  3. $\mathrm{Var}[S] = r \left( \displaystyle{\frac{1}{p} -1} \right) \left[ \mathbb{E}\left[Y^{2}\right] + \displaystyle{\left(\frac{1}{p} -1\right)} \left(\mathbb{E}\left[Y\right]\right)^{2} \right]$

  4. $\mathrm{M}_{S}(t) =\displaystyle{ \left(\displaystyle{ \frac{p}{1 - (1-p)\mathrm{M}_{Y}(t)} }\right)^{r}}$

Ejercicio

Si el riesgo $S$ en el modelo colectivo tiene distribución $\mathrm{Poisson\, Comp}(\lambda,G)$. Prueba que

  1. $\mathbb{E}[S] = \lambda \mathbb{E}[Y]$

  2. $\mathbb{E}\left[S^{2}\right] = \lambda \mathbb{E}[Y^{2}] + \lambda^{2}\left(\mathbb{E}[Y]\right)^{2}$

  3. $\mathrm{Var}[S] = \lambda \mathbb{E}\left[Y^{2} \right]$

  4. $\mathrm{M}_{S}(t) = \exp\left( \lambda \left( \mathrm{M}_{Y}(t) - 1\right) \right)$

Ejercicio

Sea $S = \displaystyle{\sum_{j=1}^{N} Y_{j}}$ el riesgo en el modelo colectivo. Si $N$ tiene distribución $\mathrm{Poisson}(\Lambda)$ en donde $\Lambda$ es una variable aleatoria con función de distribución $F_{\lambda}(\lambda)$, prueba que

  1. $\mathbb{P}(N=n) = \displaystyle{\int_{0}^{\infty} \mathrm{e}^{-\lambda} \frac{\lambda^{n}}{n!} } dF_{\Lambda}(\lambda)$

  2. $\mathbb{E}\left[S\right] = \mathbb{E}[\Lambda]\mathbb{E}[Y]$

  3. $\mathbb{E}\left[S^{2}\right] = \mathbb{E}\left[\Lambda\right] \mathbb{E}\left[Y^{2}\right] + \mathbb{E}\left[\Lambda^{2}\right]\left(\mathbb{E}[Y]\right)^{2}$

  4. $\mathrm{Var}[S] = \mathrm{Var}[\Lambda]\mathbb{E}\left( \left[Y\right]\right)^{2} + \mathbb{E}\left[Y\right] \mathbb{E}\left[Y^{2}\right]$

  5. $\mathrm{M}_{S}(t) = \mathrm{M}_{\Lambda}\left( \mathrm{M}_{Y}(t) - 1 \right)$
